*********************** Instructions to run webapp ***********************

1. Please install python3 on your system

2. Add python to environment variables

3. Navigate 'G10_HAVE' folder in terminal 

4. Create an evironment using the following command -> virtualenv HAVE_env

5. Activate the evironment using the following command  
   -> .\HAVE_env\Scripts\activate   (For Windows)
   -> source HAVE_env\bin\activate  (For Ubuntu)

6. Run command -> pip3 install -r requirements.txt

7. Run command -> python3 manage.py makemigrations

8. Run command -> python3 manage.py migrate

9. Run command -> python3 manage.py runserver

10. Click on localhost link generated after execution of previous command
